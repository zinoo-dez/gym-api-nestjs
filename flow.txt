PUBLIC WEBSITE ROUTE FLOW
Accessible without login
Goal: marketing, conversion, discovery
/
├─ /home
├─ /features
├─ /trainers
│   ├─ /trainers/:trainerId
├─ /workout-plans
│   ├─ /workout-plans/:planId
├─ /classes
│   ├─ /classes/:classId
├─ /memberships
├─ /about
├─ /contact
└─ /pricing        (optional alias of memberships)
User Flow
Home → Features → Trainers / Workout Plans → Memberships → Register


AUTH ROUTES
Public but redirect if logged in
/auth
├─ /auth/login
├─ /auth/register
├─ /auth/forgot-password
├─ /auth/reset-password
└─ /auth/verify-email
Logic
Logged-in user → redirect to /app
Admin → redirect to /admin

MEMBER (USER) ROUTES

├─ /app/attendance
├─ /app/trainers
│   ├─ /app/trainers/:trainerId
├─ /app/progress
└─ /app/settings
Member Flow
Login
 → Dashboard
 → View Membership
 → Book Class
 → Follow Workout Plan
 → Track Attendance

 ROUTE PROTECTION STRATEGY (IMPORTANT)
 Auth Guards
 PublicRoute     → for marketing pages
AuthRoute       → login/register only
ProtectedRoute  → member
